
A small Python project to manage a bank account.

 What can it do?
Create Account: Just type your name to start.

Deposit: Add money to your balance.

Withdraw: Take money out (if you have enough).

Check Balance: See how much money you have.

How to use it?
Open your terminal or IDE (like VS Code).

Run the file: python bank.py

Follow the simple menu (1, 2, 3, 4)
